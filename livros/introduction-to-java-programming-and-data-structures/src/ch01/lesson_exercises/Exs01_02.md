
1.2.1 What are hardware and software?\
- Hardware are the physical components of a computer and Software are the logical ones.

1.2.2 List the five major hardware components of a computer.\
- CPU, motherboard, memory, storage devices, input devices, output devices and communication devices.

1.2.3 What does the acronym CPU stand for? What unit is used to measure CPU speed?\
- Central Processing Unit. Hertz.

1.2.4 What is a bit? What is a byte?\
Data are stored in 0 and 1 values. Each of this value are stored in bits. 8 bits stores the minimum storage unit - the byte.

1.2.6 What unit is used to measure memory size? What unit is used to measure disk size?\
- Megabyte and gigabytes for RAM and gigabyte and terabye for disks.

1.2.7 What is the primary difference between memory and a storage device?\
A computer’s memory (RAM) is a volatile form of data storage: Any information that has been saved in memory is lost when the system’s power is turned off. Programs and data are permanently stored on storage devices.
