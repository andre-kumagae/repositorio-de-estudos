1.3.1 What language does the CPU understand?\
- Machine language.

1.3.2 What is an assembly language? What is an assembler?\
- assembly language was created in the early days of computing as an alternative to machine languages. Assembly language uses a short descriptive word, known as a mnemonic, to represent each of the
  machine-language instructions.
- an assembler is a program used to translate assembly-language programs into machine code

1.3.3 What is a high-level programming language? What is a source program?\
- A high-level language programming languages is a language easy to understand and platform independent.
- A source program is a program written in a high-level language.

1.3.4 What is an interpreter? What is a compiler?\
- A interpreter reads line by line and translates the source code to tha machine code and then executes it;
- A compiler needs to read the entire source code and then translates to the machine code and executes it; 

1.3.5 What is the difference between an interpreted language and a compiled language?\
- An interpreted language translates the code during the execution and the compiled language executes before the execution.
